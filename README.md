Использовал node.js 22.22.0 версии, на других версиях не тестировал.
Также использовал Docker для контейниризации базы данных и дальнейшего подключения к ней.

Запуск проекта
1. Подготовка базы данных:
  - Перейдите в папку проекта, куда вы скопировали репозиторий.
  - Запустите контейнер с базой данных с помощью команды:
      docker-compose up
2. Запуск сервера:
  - Откройте консоль и перейдите в папку с сервером:
      cd AngularWithAsp.Server
  - Обновите зависимости проекта:
      dotnet restore
  - Запустите сервер:
      dotnet run
3. Доступ к серверу:
  - Перейдите в браузере по адресу http://localhost:5170
4. Запуск клиента:
  - После этого откроется консоль для запуска клиента.
  - Перейдите в браузер по адресу для работы с клиентской частью http://localhost:49312
